# IPL-Analysis-Portfolio
Data Analysis of M# IPL Team Performance Analysis
ipl_analysis_chart.png
This project analyzes the win rates and head-to-head records of five major IPL teams: 
Mumbai Indians, RCB, SRH, Gujarat Titans, and Rajasthan Royals.

### Tools Used:
- **Python**: For data processing.
- **Pandas**: To filter team-specific data.
- **Matplotlib**: To visualize win statistics.

### Key Insights:
- [MI has the most wins]
